POST
​/oauth​/token
Requests an access token
Requests new access token for accessing our API resources. By default, the token expires in 6 hours.

Parameters
Try it out
No parameters

Request body

application/x-www-form-urlencoded
grantType
grantType *
The OAuth grant type. Currently, the supported grant types are client_credentials, authorization_code, and refresh_token.

clientId
clientId *
The client identifier.

clientSecret
clientSecret *
The client secret.

authorizationCode
authorizationCode
The authorization code that was returned after authorizing the application. It is only required when using the authorization code flow.

authorizationCodeVerifier
authorizationCodeVerifier
The authorization code verifier that was returned in the user code request. It is only required when using the authorization code flow. The token request will fail in case the code verifier is not present or in case it does not match the one returned in the user code request.

refreshToken
refreshToken
The refresh token that was returned after requesting an access token. It is only required when using the refresh_token grantType. Therefore, refresh tokens are only available when using distributed apps in the authorization code flow.
