feat(automations): Implementa salvamento e teste para relatórios semanais/mensais

Este commit corrige um bug onde as configurações de automação para relatórios semanais e mensais não eram salvas nem podiam ser testadas.

Modificações:
- `backend/controllers/whatsappConfigController.js`: As funções `getInstanceConfig` e `updateInstanceConfig` foram atualizadas para carregar e salvar corretamente os dados dos relatórios semanal e mensal, que antes eram ignorados.
- `frontend/services/automationService.js`: Adicionadas as funções `testWeeklyReport` e `testMonthlyReport` para se comunicar com os endpoints de teste do backend.
- `frontend/components/AutomationTester/index.js`: O componente de teste foi atualizado para invocar as novas funções de serviço, habilitando o botão "Enviar Teste" para todos os tipos de relatório.
